
# Meu Portfolio

Portfolio pessoal em HTML, CSS e JavaScript com foco em apresentacao profissional, servicos e contato.

## Visao geral
- Pagina única com seções: Home, Sobre, Habilidades, Qualificacões, Serviços e Contato.
- Layout responsivo com ajustes especificos para mobile.
- Navegacao por ancoras e CTA para LinkedIn.

## Estrutura do projeto
```
index.html
assets/
	css/
		style.css
	js/
		main.js
	img/
private/
	img/
	pdf/
```

## Como visualizar
1. Abra o arquivo index.html no navegador.
2. Opcional: use uma extensao de Live Server para recarregar automaticamente.

## Personalizacao rapida
- Texto e links: edite index.html.
- Estilos e responsividade: edite assets/css/style.css.
- Scripts de interacao: edite assets/js/main.js.
