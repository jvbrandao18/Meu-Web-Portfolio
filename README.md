# Meu Portfólio

Site de portfólio pessoal com layout responsivo, seções de apresentação, habilidades, qualificações, serviços e contato.

## Visão Geral
- Página única (SPA) em HTML/CSS/JS
- Layout responsivo com mobile-first
- Tema claro/escuro
- Seções com conteúdo organizado e foco em usabilidade

## Tecnologias
- HTML5
- CSS3 (Flexbox + Grid)
- JavaScript (vanilla)
- Unicons

## Estrutura do Projeto
```
.
├── assets/
│   ├── css/
│   │   └── style.css
│   ├── img/
│   ├── js/
│   │   └── main.js
│   └── pdf/
├── index.html
└── README.md
```

## Funcionalidades
- Navegação com destaque da seção ativa
- Menu responsivo (mobile)
- Alternância de tema (claro/escuro)
- Seção de serviços com expansão suave ao passar o mouse
- Links de contato (mailto)
